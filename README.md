# node-excel-generator

Generates Excel (`.xlsx`) files from JSON data using Node.js and the [`xlsx`](https://www.npmjs.com/package/xlsx) library.

## Features
- Convert arrays/JSON to an Excel worksheet
- Create workbooks with named sheets
- Set basic column widths
- Save to disk

## Prerequisites
- Node.js 16+  
- npm

## Setup
```bash
npm init -y
npm i xlsx
